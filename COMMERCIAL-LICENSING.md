# Commercial Licensing

The public repository uses a split non-commercial licence model:

- documentation, design text, blueprints, and public reports: `CC-BY-NC-4.0`
- source code and reproducibility tooling: `LICENSE-CODE.md`

Commercial use is not granted by the public licence. Companies, investors,
foundries, product teams, or other commercial users need a separate written
commercial licence or collaboration agreement before using the design,
documentation, reports, code, or simulator output for commercial purposes.

Commercial discussions should treat the current repository as a simulation-only
review package. It is not a representation that the design is build-ready,
prototype-ready, foundry-ready, tapeout-ready, or hardware-validated.

Any commercial evaluation should request, at minimum:

- independent technical review of the simulator and report assumptions
- foundry PDK and compact-model review
- DRC/LVS plan
- foundry-calibrated S-parameters
- source, detector, package, and control evidence
- measured testchip or device data
- fault-tolerance and decoder review

Patent, trademark, and hardware-manufacturing rights are not granted by
the public repository.
